Some W3C Documents in EPUB3 formats

This is a list of some W3C documents converted into EPUB3 format. These versions have no formal standing, and are here just for convenience and as examples for EPUB3 files. They are nevertheless useful if, for example, a W3C Recommendation consists of several documents (e.g., RDFa); these examples below “bind” such documents together into one book that can also be downloaded for off-line reading.

Currently available:

  • “SPARQL 1.1”; W3C Recommendation, 21 March, 2013
  • “RDFa — RDF in Attributes”; W3C Recommendation, 22 August, 2013
  • “JSON for Linking Data (JSON-LD)”; W3C Recommendation, 16 January, 2014
  • “RDF 1.1 Core”: RDF 1.1 Primer, Concepts, Semantics; W3C Recommendation, 25 February, 2014
  • “RDF 1.1 Serialization Syntaxes”: Turtle family, JSON-LD, RDFa, JSON, RDF/XML; W3C Recommendations, 2013 and 2014
  • “CSV on the Web”: W3C Recommendations, 2015, and Working Group Notes, 2016
  • “Web Annotations”: W3C Recommendations and Working Group Notes, 2017
  • “ODRL 2.2”: W3C Recommendations, 2018
  • “EPUB 3.3”: W3C Recommendation, 2023

The books were produced by several generations of experimental tools to convert individual W3C Technical Documents (authored with respec), as well as collections thereof, into EPUB3. The current tool is r2epub.
